# Passos para rodar a aplicação

# Clonar repositório

- git clone git@github.com:tiagobr21/conectus-project.git

# Entrar na pasta do projeto

- cd conectus-project

# Instalar as dependências do projeto

- npm install

# Inicializar a aplicação

- npm run dev

# Acessar no Navegador

- http://localhost:5173
